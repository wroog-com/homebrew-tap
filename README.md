# Wroog Homebrew Tap

Homebrew formulae for [Wroog](https://github.com/wroog-com) tools.

## Install

```sh
brew install wroog-com/tap/demi
```

`demi` is a terminal-native project lifecycle companion — see
[wroog-com/demiurge](https://github.com/wroog-com/demiurge).

Formulae in this tap are generated and updated automatically by GoReleaser
when a release ships; they are not edited by hand.
